# Deployment Instructions for Aaditi Physio

1. Create repo: Go to [github.com/new](https://github.com/new) and name the repository **`aaditi-physio-site`**.
2. Initialize and push your local `aaditi-physio-site` folder to this new repository.
3. Go to **Settings → Pages**.
4. Set Source to **Deploy from a branch**.
5. Select the **`main`** branch (or `master`) and click **Save**.
6. Your live URL will be: **`https://basavarajbillur000.github.io/aaditi-physio-site/`**
